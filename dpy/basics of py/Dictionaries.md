
a **dictionary** is an unordered (but preserved by insertion order since Python 3.7), mutable collection of **key-value pairs**.

eg:
```
# Creating a dictionary
user_profile = {
    "username": "coder_99",
    "followers": 1250,
    "is_active": True
}
```

```
d = {"a": "apple", "b": "ball"}
print(d)

```

## Adding elements:

```
d = {"a": "apple", "b": "ball"}
print(d)
d["c"] = "cat"
print(d)
```
