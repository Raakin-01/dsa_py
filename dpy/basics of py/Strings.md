```

PS C:\Users\hp\OneDrive\Desktop\dsa_py> python .\pyBasics.py
```

## Format:

```
name = "Raakin"
fav_num = 20
print("My name is {0} and my favourite number is {1}.".format(name, fav_num))

```
 Output:
```
PS C:\Users\hp\OneDrive\Desktop\dsa_py> python .\pyBasics.py
My name is Raakin and my favourite number is 20.
```
